## 🆘 Troubleshooting

### Login doesn't work?
1. **Clear localStorage:**
   - Open console (F12)
   - Type: `localStorage.clear()`
   - Press Enter
   - Reload page

2. **Hard refresh:**
   - Press: `Ctrl + F5`

3. **Check console:**
   - Open F12
   - Look for errors
   - Should see "✅ Admin user created"

### Already logged in?
- System auto-redirects to dashboard
- To logout: Clear localStorage, reload
