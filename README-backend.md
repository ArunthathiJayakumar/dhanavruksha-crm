# DHANAVRUKSHA CRM - Backend Setup

## 🚀 Separate Backend for Custom Domain

### **Problem Solved:**
- ✅ **Frontend on GitHub Pages** with custom domain
- ✅ **Backend on Vercel** for data persistence
- ✅ **CORS enabled** for cross-origin requests
- ✅ **Real database** functionality

### **🔧 Backend Deployment:**

#### **1. Create New Repository:**
1. Go to GitHub → Create new repository
2. Name: `dhanavruksha-crm-api`
3. Clone: `git clone https://github.com/ArunthathiJayakumar/dhanavruksha-crm-api.git`
4. Copy `api/` folder to this repository
5. Add `vercel.json` and `package.json`

#### **2. Deploy to Vercel:**
1. Go to Vercel.com
2. Import `dhanavruksha-crm-api` repository
3. Deploy → Get URL: `https://dhanavruksha-crm-api.vercel.app`

#### **3. Update Frontend:**
- Frontend API calls go to Vercel backend
- Custom domain works on GitHub Pages
- Data persists in backend database

### **🌐 Architecture:**

```
GitHub Pages (Frontend)
    ↓ API Calls
Vercel (Backend API)
    ↓ Data Storage
Database/Storage
```
